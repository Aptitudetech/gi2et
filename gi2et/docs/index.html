Get Github Issues into ERPNext Tasks

One way pull and update of Github Issues into ERPNext Tasks

Pull Github Issues into Project Tasks. Update existing Issues into existing Tasks

Install

From your site

To install this app, login to your site and click on "Installer". Search for Gi2Et and click on "Install"

Using Bench

Go to your bench folder and setup the new app

$ bench get-app gi2et https://github.com/gaurav-naik/gi2et
$ bench new-site testsite
$ bench --site testsite install-app gi2et

Login to your site to configure the app.
